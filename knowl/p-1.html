A relation is a collection of pairs.

Example 1.1.1.

\(\{(1,x),(3,t),(1,y),(4,y)\}\) is a relation.
The set of all first entries of the pairs in a relation is called the domain, and the set of all second entries is called the range. We will denote them by \(D\) and \(R\text{,}\) respectively.

Example 1.1.2.

  1. The relation \(\{(1,x),(3,t),(1,y),(4,y)\}\) has domain \(D=\{ 1,3,4\}\) and range \(R=\{x,t,y\}\) (remember that when listing the elements of a set, we never repeat the same element twice).
  2. The relation \(\{(0,c),(1,d),(2,c),(3,e),(4,a),(5,f)\}\) has domain \(D=\{0,1,2,3,4,5\}\) and range \(R=\{a,c,d,e,f\}\)
  3. The relation \(\{ (1,1), (2,1), (3,1), (2,1)\}\) has domain \(D=\{1,2,3\}\) and range \(R=\{1\}\text{.}\)
